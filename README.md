# GlassFinance — Трекер Личных Финансов

Интерактивное веб-приложение для отслеживания доходов и расходов, разработанное на Django с использованием базы данных SQLite и современного темного интерфейса в стиле Glassmorphism.

## 🌟 Функционал приложения
1. **Аутентификация и безопасность**:
   - Регистрация пользователей с автоматической проверкой уникальности имени пользователя;
   - Безопасный вход с сессиями и хешированием паролей (PBKDF2);
   - Выход из аккаунта.
2. **Дашборд (Bento Grid)**:
   - Отображение текущего баланса, общих доходов и расходов;
   - **Интерактивный график (Chart.js)**: круговая диаграмма распределения трат по категориям;
   - **Форма быстрого добавления транзакций** (сумма, тип, категория, дата, описание);
   - **Список последних транзакций** с возможностью удаления.
3. **Асинхронные операции (AJAX / Fetch API)**:
   - Добавление и удаление транзакций происходит мгновенно **без перезагрузки страницы**;
   - Баланс, доходы, расходы и график перерисовываются на лету.

---

## 🛠️ Технологический стек
* **Backend**: Python 3.12, Django 5.x
* **Database**: SQLite3
* **Frontend**: HTML5 (Django Templates), Vanilla CSS, Vanilla JavaScript, Chart.js (CDN), Lucide Icons (CDN)

## 🚀 Инструкция по запуску

### 1. Подготовка окружения
Клонируйте репозиторий и перейдите в его корневую директорию.
Создайте и активируйте виртуальное окружение:
```bash
python -m venv .venv
# Для Windows (PowerShell):
.venv\Scripts\activate
# Для Linux/macOS:
source .venv/bin/activate
```

### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 3. Применение миграций и наполнение базы данных
```bash
python manage.py migrate
python manage.py seed_categories
```
*Примечание: команда `seed_categories` также создаст демонстрационного пользователя для быстрой проверки.*

🔑 **Данные для демо-аккаунта:**
* **Username:** `demo`
* **Password:** `password123`

### 4. Запуск сервера разработки
```bash
python manage.py runserver
```
Приложение будет доступно по адресу [http://127.0.0.1:8000/](http://127.0.0.1:8000/).
